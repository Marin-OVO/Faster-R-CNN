# 遥感小目标检测实验代码说明
//本仓库为课程实验代码仓库，主要用于遥感影像小目标检测相关方法的复现与对比实验，整体实现基于PyTorch框架，并围绕Faster R-CNN + FPN模型实验.
//需要特别说明的是: **数据集不包含在本仓库中，且已被主动移除**.
//请在使用本代码前自行准备并配置相应的数据集.


## 一、运行环境
本实验代码在Python 3.11环境下开发与测试，推荐使用Conda或虚拟环境进行依赖管理.

* Python: 3.11.7
* 操作系统: Windows
* GPU: NVIDIA GPU（推荐）
* CUDA: 12.1


## 三、依赖库说明
以下库为代码正常运行与训练所必需的核心依赖，缺失将导致程序无法启动或训练失败:

* torch == 2.1.0+cu121  
* torchvision == 0.16.0+cu121 
* numpy == 1.26.4
* opencv-python == 4.12.0.88
* pycocotools == 2.0.10
* albumentations == 2.0.8
* matplotlib == 3.7.5
* tqdm == 4.67.1
* pyyaml == 6.0.1
* pandas == 2.0.3 
* scipy == 1.10.1 
* scikit-learn == 1.3.2
* pillow == 9.5.0

1. 创建并激活Python虚拟环境
2. 安装上述**核心必须依赖库**
